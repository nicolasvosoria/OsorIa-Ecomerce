# Project AGENTS

## Project Standards

- Prefer strong typing and explicit domain models.
- Keep architecture modular and split responsibilities early.
- Apply SOLID where it improves maintainability.
- Favor DRY through typed helpers/constants, not copy/paste.
- Keep files small, readable, and easy to change.
- Separate reusable constants/types from view-heavy modules.
